Directions for Use
1. At the main folder of the of the project, right click and add the entire folder and subfolders to path.
2. Run the codec from >encoder>encoder_main.m
3. Example performance metrics and various works spaces can be accessed from the performance folder.
4. Utilities and tools are available in the common folder
5. The >decoder folder contains the tools used in the frame reconstruction but the bitstream decoder is not yet implemented.
